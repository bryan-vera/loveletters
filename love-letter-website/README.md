# Love Letter Website

This project is a romantic-themed love letter website that simulates the experience of receiving a heartfelt letter. The website features an interactive envelope that, when clicked, reveals a love letter being typed out in real-time. Users can also respond to the letter with their own message.

## Project Structure

```
love-letter-website
├── src
│   ├── index.html          # Main HTML document
│   ├── css
│   │   ├── style.css       # Main styles for the website
│   │   └── animations.css   # CSS animations for envelope and typing effects
│   ├── js
│   │   ├── main.js         # Entry point for JavaScript functionality
│   │   ├── typewriter.js    # Logic for the typewriter effect
│   │   └── envelope.js      # Manages envelope interactions
│   └── assets
│       └── fonts
│           └── fonts.css    # Custom fonts for the website
├── package.json             # Configuration file for npm
└── README.md                # Documentation for the project
```

## Features

- **Interactive Envelope**: Click the envelope to start the typing animation of the love letter.
- **Typewriter Effect**: The letter appears as if it is being typed in real-time, enhancing the romantic experience.
- **Response Input**: Users can respond to the letter with their own message, featuring a send or not send option.

## Getting Started

1. **Clone the Repository**:
   ```bash
   git clone https://github.com/yourusername/love-letter-website.git
   cd love-letter-website
   ```

2. **Install Dependencies** (if any):
   ```bash
   npm install
   ```

3. **Open the Website**:
   Open `src/index.html` in your web browser to view the love letter website.
